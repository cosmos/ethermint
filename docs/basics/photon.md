## Base Denemination

Ethermint uses Atto Photon as the base Denomination to maintain parity with Ethereum.

```
1 photon = 1×10⁻¹⁸ APhoton
```

This matches Ethereum denomination of

```
1 ETH = 1x10⁻¹⁸ wei
```
